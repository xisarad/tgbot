# Учебный бот

Этот бот повторяет все текстовые сообщения (эхо-бот).

После клонирования необходимо создать и активировать виртуальное окружение, например, в linux и mac:
python3 -m venv env
source env/bin/activate

## Установка:

pip install -r requirements.txt

Скопировать config.py.example в config.py и изменить его

## Запуск:

python main.py

## О версиях

Проект использует [семантическое версионирование](https://semver.org/).
