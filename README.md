# Учебный бот

После клонирования необходимо создать и активировать виртуальное окружение, например, в linux и mac:
python3 -m venv env
source env/bin/activate

## Установка:

pip install -r requirements.txt

Скопировать config.py.example в config.py и изменить его

## Запуск:

python main.py
## Работа с переменными окружения

После обновления бот использует переменные окружения для конфигурации.

### Быстрый запуск:
```bash
docker run -e token="ВАШ_ТОКЕН" telegram-bot


## Запуск с помощью Docker Compose

Для запуска бота используйте Docker Compose:

1. Создайте файл `.env` в корне проекта и укажите ваш токен:
   ```
   token="ВАШ_ТОКЕН"
   ```

2. Запустите контейнер:
   ```bash
   docker compose up -d
   ```

**Важно:**
- `Dockerfile` - для создания образа
- `compose.yml` - для запуска контейнера из образа
- Эти операции могут выполняться на разных машинах

В реальной работе образ собирается на сервере сборки и помещается в registry,
а затем запускается на отдельном сервере.
